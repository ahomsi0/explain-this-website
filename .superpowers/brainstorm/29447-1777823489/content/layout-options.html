Desktop Layout — How should we use the extra space?

Right now the content column is capped at ~1180px and leaves large empty margins on wide monitors. Pick the approach that appeals most.

A — 3-Column Card Grid

Keep the sidebar. On large screens (≥1280px), cards switch from 2-col to 3 columns so content fills the page. Minimal change, big visual improvement.

✓ Easiest to implement  ✓ Keeps existing structure

B — Tab Bar + Full-Width Grid

Replace the left sidebar with a horizontal tab bar across the top. The full width is now content — 3-column grid on desktop, fluid on mobile.

✓ Max content space  ⚠ More redesign work

C — Sidebar + Summary Panel

Keep sidebar on the left, main cards in center, and add a persistent right panel on wide screens showing quick-glance scores and top issues.

✓ Always-visible summary  ⚠ Significant new component
